1. 합성곱 신경망의 구성 요소
1) 합성곱(convolution): 입력 데이터에서 유용한 특성만 드러나게 하는 것
 (1) 합성곱 신경망(convolutional neural): 합성곱 방식을 사용하는 신경망 모델
 (2) 커널(kernel, 필터, filter): 합성곱 신경망의 뉴런
 (3) 특성 맵(feature map): 합성곱 계산을 통해 얻은 출력

2) 패딩
 (1) 패딩(padding): 입력 배열의 주위를 가상의 원소(0)로 채우는 것
	>> 출력 배열의 크기를 키울 수 있음
 (2) 세임 패딩(same padding): 입력과 특성 맵의 크기를 동일하게 만들기 위해 패딩
 (3) 밸리드 패딩(valid padding): 패딩 없이 입력 배열에서만 특성 맵을 만드는 경우

3) 스트라이드(stride): 필터가 이동할 때의 방식
	>> 기본은 1로 한칸씩 이동

4) 풀링
 (1) 풀링(pooling): 합성곱 층에서 만든 특성 맵의 가로세로 크기를 줄임
	>> (k, k, n) >pooling> (k`, k`, n)      * k > k`
	*곂치지 않도록 추출함
 (2) 최대 풀링(max pooling): 특성 맵에서 가장 큰 값을 추출
 (3) 평균 풀링(average pooling): 특성 맵에서 평균값을 추출

		   가중치가 시각적인 패턴을 학습하는지 확인
특성 맵 시각화: 합성곱 층의 활성화 출력을 이미지로 그리는 것
		    각 필터가 이미지의 어느 부분을 활성화시키는지 확인
